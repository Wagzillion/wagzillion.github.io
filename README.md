# Bryan Wagner — Portfolio Site

Personal portfolio and professional landing page.

## Files

- `index.html` — Main site
- `style.css` — Stylesheet (dark theme)
- `main.js` — Nav toggle + scroll animations
- `resume.pdf` — Place your resume PDF here (rename to `resume.pdf`)

## Deploying to GitHub Pages

1. Create a new GitHub repository named `bryanwagner.github.io`
   (must match your GitHub username exactly for the root domain URL)

2. Upload all files from this folder to the repository root

3. Go to **Settings → Pages**

4. Under **Source**, select `Deploy from a branch` → `main` → `/ (root)`

5. Click **Save** — your site will be live at `https://bryanwagner.github.io`
   within a few minutes

## Adding Your Resume PDF

Rename your resume file to `resume.pdf` and place it in the same folder
as `index.html`. The "Download Resume" button and contact page link will
pick it up automatically.

## Custom Domain (Optional)

To use a custom domain like `bryanwagner.dev` (~$12/year via Google Domains
or Namecheap):

1. Purchase the domain
2. In GitHub Pages settings, enter the custom domain
3. Add a CNAME DNS record pointing to `bryanwagner.github.io`
4. Check "Enforce HTTPS"
